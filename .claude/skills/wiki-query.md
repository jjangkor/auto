# wiki-query

위키에 질문하고 답변을 생성합니다.

## 사용법
`/wiki-query [질문]`

## 워크플로우

1. **인덱스 탐색**: `wiki/index.md` 와 `wiki/index_summary.md` 를 읽어 관련 페이지 파악

2. **페이지 읽기**: 관련 위키 페이지들을 읽고 정보를 종합

3. **답변 생성**: 위키 내용을 기반으로 답변 작성
   - 출처 페이지를 `[[위키링크]]`로 인용
   - 위키에 없는 정보는 명시적으로 구분
   - 필요시 다양한 형식 사용 (표, 비교, 타임라인 등)

4. **위키 반영 판단**: 답변이 위키에 보존할 가치가 있는지 사용자에게 확인
   - 가치 있는 분석 → `wiki/analysis/` 에 페이지로 저장
   - 새 인사이트 → 관련 기존 페이지에 통합
   - 일회성 답변 → 위키에 저장하지 않음

5. **로그 기록**: `wiki/log.md` 에 쿼리 기록 append

## 답변 원칙
- 위키에 있는 내용 기반으로 답변 (근거 명시)
- 위키에 없는 내용은 "위키에 아직 없는 정보" 로 표시
- 모순이 있으면 양쪽 모두 제시
- 지식 갭 발견 시 추가 소스 제안
